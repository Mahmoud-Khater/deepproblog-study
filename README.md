# DeepProbLog Study Materials

This repository contains study materials and examples for DeepProbLog, a neural probabilistic logic programming language.

## Contents

- `Programming_In_DeepProlog.txt` - Cleaned academic paper on DeepProbLog
- `deepproblog_examples.pl` - Prolog code examples extracted from the paper
- `clean_document.py` - Python script used to clean the PDF extraction

## About DeepProbLog

DeepProbLog is a neural probabilistic logic programming language that incorporates deep learning by means of neural predicates. It integrates:

- **Logical reasoning** with neural networks
- **Symbolic and subsymbolic** representations and inference
- **Probabilistic (logic) programming**
- **Deep learning** from examples

## Examples Included

1. **MNIST Single-Digit Addition (T1)** - Basic digit classification and addition
2. **MNIST Multi-Digit Addition (T2)** - Multi-digit number operations
3. **ProbLog Alarm Network** - Bayesian network example
4. **Forth Addition (T5)** - Program sketching with neural holes
5. **Forth Sorting (T6)** - Bubble sort with neural swap decisions
6. **CLUTRR Family Relations (T11)** - Natural language reasoning
7. **Coin Classification (T8)** - Distant supervision example
8. **Noisy MNIST Addition (T4)** - Handling noisy labels

## Requirements

To run the DeepProbLog examples, you need:
- DeepProbLog framework: https://bitbucket.org/problog/deepproblog
- ProbLog for probabilistic logic programming
- PyTorch for neural network components
